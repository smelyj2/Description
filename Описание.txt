1)card - Вёрстка карточки с игры "энергосеть".

2)chat - Вёрстка чата по примеру мобильной версии где всё сделано на css и html кроме скрепки.

3)RegForm - Вёрстка регистрационной формы и создание валидации этой формы.

4)remail1, remail2 и remail3 - Табличная вёрстка ремейлов.

5)site1 и site2 - Вёрстка статических сайтов. 

6)Sklad(storehouse)- Вёрстка статичного сайта. Создание статического слайдера. Написание валидатора регистрационной формы.

7)tabs - Вёрстка и создание элемента управления на JavaScript(jQuery) для создания\редактирование\удаления новых вкладок.

8)PhoneCategory - Моё первое знакомство с Ангуляром и создания на нём динамического сайта и применение MVC модели в нём.

9)authorization - Вёрстка авторизации. Написание капчи, валидация регистрационных данных, отправка регистрационных данных на сервер используя 
Ajax запросы. Написание серверной части используя PHP ,а именно: приём данных с клиентской стороны, обработка данных, внесение данных в MySql таблицу.
Написание самой авторизации на PHP , при этом если пользователь успешно авторизирован, то ему доступен функционал добавления фотографий которые тоже
запоминаются в таблице MySql, так при выходе с сайте и новом входе на сайт сохраняется его фотография, таким способом можно расширить функционал для 
авторизированных пользователей и организовать запоминание всех его предыдущих действий. 

10)aimProSoft-test - Адаптивная вёрстка сайта по трём макетам для пк,планшетов и мобильных устрйоств. Создание адаптивного слайдера, выпадающего списка на
JavaScript(jQuery).

11)RiffPoint-test - Адаптивная вёрстка сайта используя Bootstrap, fontAwesome иконок, спрайтовых изображений. Свой собственный дизайн для отображения на
планшетных и мобильных устройствах. Создание слайдера на Bootstrap.

12)add-comments-site - Адаптивная вёрстка сайта используя Bootstrap. Написание на JavaScript добавления комментариев имени пользователя и текущей даты в 
нужном формате.Также доступна возможность добавления комментариев используя комбинацию клавиш Ctrl+Enter.

13)RegForm2 - Вёрстка формы,создание стурктуры БД. Валидация введёных данных на клиентской стороне без перезагрузки страницы, подгрузка данных из БД 
в взаимосвязный список "select" на форме, обработка данных на серверной стороне. Реализация регистрационной формы.

14)internetStore - Адаптивная вёрстка интернет магазина состоящего из четырёх страниц. С возможностью детально посмотреть картинку выбранного товара(на странице
item_details_page).C корзиной покупок  на LocalStorage.
